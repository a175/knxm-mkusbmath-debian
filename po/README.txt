If you want to give a translation of a new language, e.g. ja,
the do the following:

1. Run the following command:
       make ja.pox
2. We have a pox file like ja.pox.
3. Edit the pox file.
4. Run the following command:
       make



If the scriptfile is updated,
then do the following:

1. Run the following command:
       make
   or the command like:
       make ja.pox
2. We have the updated pox file.
3. Edit the pox file.
4. Run the following command:
       make
